Projects

Tetris and AI to Play Tetris

I programmed tetris in javascript using a html canvas.
For the AI, I used a depth-first search transversal of the game board to test all possible moves,
and weighed certain parameters which consitute a "good" move to decide which move is best.
The parameters I used were summed height, full rows, empty tiles with a non-empty tile above them, and height variance
By using a hash set I was able to more than quadruple the performance of the AI compared to a brute-forced depth-first search.
Additionally, by not starting the depth-first search until the moving piece reached a non-empty row, I improved the speed of the algorithm two-fold.

W : Rotate Clockwise

A : Move Left

S : Rotate Counterclockwise

D : Move Right

+ : Double Game Speed

- : Halve Game Speed
